    Installation

    Welcome to the Air Force Cadet Career Problem (AFCCP) installation guide. This guide walks you through cloning the repository, installing dependencies, and running the model.

    üöÄ 1. Clone the Repository

    First, clone the AFCCP repository from GitHub:

    git clone https://github.com/dglaird/afccp.git
cd afccp

    Make sure you have Git installed.

    üêç 2. Set Up Your Python Environment

    We recommend using conda or venv to isolate your Python environment.

    Option A: Using conda

    conda create -n afccp python=3.8 -y
conda activate afccp

    Option B: Using venv

    python3 -m venv afccp-env
source afccp-env/bin/activate  # On Windows: afccp-env\Scripts\activate

    üì¶ 3. Install Dependencies

    Install all required Python packages using pip:

    pip install -r requirements.txt

    This includes packages like: - pandas, numpy ‚Äî for data wrangling - pyomo ‚Äî for optimization modeling - matplotlib ‚Äî for visualizations - mkdocs, mkdocs-material ‚Äî for documentation (optional)

    üõ† 4. Run an Example

    Navigate to the main model script (adjust path if needed):

    python afccp/main.py

    This should execute a full cadet-to-AFSC assignment using the default input data.
